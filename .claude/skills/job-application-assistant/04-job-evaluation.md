# 04 — Job Evaluation Criteria

## Location filter (hard filter)

- **Pass:** Fully remote roles (any employer location)
- **Pass:** In-office or hybrid roles based in Yerevan, Armenia
- **Fail / flag for review:** In-office roles requiring relocation elsewhere,
  unless the role is exceptional enough to reconsider — flag rather than auto-reject

## Role-level scope

Score both of these as in-scope, don't over-index toward one:
- **Lateral:** Financial Controller (or equivalent title)
- **Stretch:** Finance Director, Head of Finance, Senior Finance Manager

## Skill match — weight these highly

1. Advanced Excel / Power Query / financial modeling
2. IFRS application (especially IFRS 9) — DipIFR + hands-on multi-entity IFRS
   close give real credibility here
3. Financial reporting, closing, and consolidation across multi-entity groups
4. Budgeting, forecasting, variance / plan-vs-actual analysis
5. Large-scale or trading/commodity-adjacent reporting environments
6. ERP/systems fluency — direct experience with Oracle NetSuite, SAP, 1C,
   QuickBooks, Zoho Books, Bitrix24; comfortable learning a new stack
7. Data/analytics tools — Stata, R, Python, Bloomberg, Thomson Reuters

## Motivation filters — weight positively

- Explicit growth path toward more management/leadership responsibility
- Roles that involve learning a new tool, system, or reporting stack
- Roles where technical IFRS depth is valued, not just "closes the books"

## Deprioritize

- Roles that are purely process/administrative with no technical or IFRS component
- Roles with no stated growth path
- Roles requiring in-office presence outside Yerevan without remote flexibility

## Industry

Open — verified experience spans oil & gas/trading (Novatek), AgriTech
(Greeneration), EdTech (Yaizy), SaaS (Smartcat, Readymag, Nicity), and IT
consulting (FJX). No industry exclusions; company type and role content matter
more than sector.
